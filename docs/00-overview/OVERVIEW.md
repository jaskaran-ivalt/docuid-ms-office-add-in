# DocuID Project Overview

## 1. Introduction

**DocuID** is a Microsoft Office Add-in designed to provide a seamless and highly secure method for accessing sensitive documents. It integrates biometric authentication via the user's mobile device, ensuring that only authorized individuals can open and view confidential files directly within Word, Excel, and PowerPoint.

## 2. Mission & Goals

- **Mission**: To eliminate the risk of unauthorized document access by replacing traditional passwords with robust, user-friendly biometric verification.
- **Goals**:
  - Provide a frictionless user experience for secure document retrieval.
  - Ensure enterprise-grade security and compliance.
  - Offer seamless integration with the Microsoft Office ecosystem.

## 3. Key Features

- **Biometric Login**: Authenticate using fingerprint or face recognition on a registered mobile device.
- **Secure Document Vault**: Access a centralized, secure repository of documents.
- **Office Integration**: Open documents directly in the Office application without leaving the context.
- **Cross-Platform**: Works across Office on the web and desktop.

## 4. Target Audience

- **Primary**: Enterprises in regulated industries (finance, healthcare, legal) that handle sensitive information.
- **Secondary**: Individuals and small businesses seeking to enhance their document security.
